3. El ritmo: las semicorcheas

¿Qué son las semicorcheas?

Las semicorcheas son figuras rítmicas muy rápidas que se utilizan para crear movimiento y energía en la música. Aparecen tanto en la música clásica como en la música actual.


Equivalencia

Habíamos visto que las figuras rítmicas tienen diferentes duraciones, pero muchas veces equivalen entre sí. Esto significa que varias figuras pequeñas pueden ocupar el mismo tiempo que una figura más grande.

Habíamos dicho que una negra dura un pulso completo. Dentro de ese pulso pueden caber dos corcheas o cuatro semicorcheas.
Por eso decimos que:

-1 negra = 2 corcheas
-1 negra = 4 semicorcheas
-1 corchea = 2 semicorcheas


Estas equivalencias ayudan a entender mejor el ritmo y a mantener el pulso cuando tocamos o escuchamos música. Por ejemplo, si el pulso va muy rápido, es más probable que aparezcan semicorcheas para dar sensación de movimiento y energía.

Hazlo tú mismo/a

ACTIVIDAD

Para entender mejor cómo medimos las semicorcheas, podemos probar con palabras. Como una negra son 4 semicorcheas, vamos a buscar una palabra que tenga 4 sílabas, como «chocolate».

  • Vamos a escuchar un metrónomo marcando el ritmo. Siguiendo el pulso, tenemos que decir «cho-co-la-te».

                                                       

Puedes usar palmas y/o percusión corporal para practicarlo.
